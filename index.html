Multivariate Calculus

Double Integration

A double integral adds infinitely many small values over a two-dimensional region. It is used to calculate area, volume under a surface, mass of a lamina, average value, and bounded-region problems.

What You Will Learn

Set up and evaluate double integrals on rectangular regions, solve area bounded by curves, understand limits with infinity, and practice with guided questions and tests.

Main Formula

∫∫R f(x,y) dA

When f(x,y) = 1, the double integral gives the area of region R.

Double Integration Theory

1. Meaning

The double integral of f(x,y) over region R is the accumulated value of the function over that complete region.

∫∫R f(x,y) dA

2. Area Bounded by Curves

If the question asks for area bounded by curves, use f(x,y)=1. For curves y=g1(x) and y=g2(x):

Area = ∫ab ∫g1(x)g2(x) 1 dy dx

3. With Limits

If limits are given, directly place them in the integral and solve inside to outside.

Example: 0 <= x <= 2, x <= y <= 4.

4. Without Limits

If limits are not given, enter only the boundary graphs. The solver finds their intersection points, builds the limits, and evaluates the double integral over that region.

∫ab ∫lower graphupper graph f(x,y) dy dx

5. Infinity Limits

Type inf, infinity, +inf, or -inf in a limit box. The numeric solver approximates infinity using a large cutoff and clearly marks it as an improper integral approximation.

6. Choosing Order

Use dy dx when vertical strips are simple. Use dx dy when horizontal strips are simple.
